## 🎨 Brand Color Palette

### Primary Colors (Updated)
```css
/* Deep Black - Neutral Core & Versatility */
--artup-primary: #0D0D0D;
--artup-primary-rgb: rgb(13, 13, 13);
--artup-primary-hsl: hsl(0, 0%, 5%);
--artup-primary-hex: #0D0D0D;

/* Electric Blue - Creativity & Energy */
--artup-accent: #3B82F6;
--artup-accent-rgb: rgb(59, 130, 246);
--artup-accent-hsl: hsl(217, 91%, 60%);
--artup-accent-hex: #3B82F6;

/* Bright Purple - AI & Innovation */
--artup-secondary: #8B5CF6;
--artup-secondary-rgb: rgb(139, 92, 246);
--artup-secondary-hsl: hsl(262, 87%, 66%);
--artup-secondary-hex: #8B5CF6;
```

**Primary Color Selection Rationale:** The neutral Deep Black `#0D0D0D` is chosen as the core anchor to maximize adaptability across light/dark themes, avoid early visual lock-in, and provide a stable base for future accent system growth (Electric Blue / Bright Purple). This establishes a scalable semantic token layer rather than prematurely centering the identity on a chromatic hue.

### Neutral Colors
```css
/* Text & Interface */
--artup-dark: #1F2937;
--artup-gray: #6B7280;
--artup-light-gray: #F3F4F6;
--artup-white: #FFFFFF;
```

### Gradient Combinations (Updated)
```css
/* Primary Gradient (updated) */
background: linear-gradient(135deg, #0D0D0D 0%, #3B82F6 100%);

/* AI Gradient */
background: linear-gradient(135deg, #3B82F6 0%, #8B5CF6 100%);

/* Full Spectrum (updated) */
background: linear-gradient(135deg, #0D0D0D 0%, #3B82F6 50%, #8B5CF6 100%);
```

### Wordmark Monochrome Usage
The core wordmark is restricted to monochrome presentation:
- Light backgrounds: `#0D0D0D`
- Dark / accent backgrounds: `#FFFFFF`
- Accent colors (Blue / Purple) are reserved for UI elements, not the base wordmark fill.

Rationale: Stability of recognition, accessibility contrast control, and future flexibility for introducing a separate symbol/logomark.

## 🔤 Typography Guidelines

### Recommended Font Families

#### Primary (Headings)
```css
font-family: "Inter", "SF Pro Display", "Segoe UI", sans-serif;
```

#### Secondary (Body Text)
```css
font-family: "Inter", "SF Pro Text", "Segoe UI", system-ui, sans-serif;
```

#### Monospace (Code/Technical)
```css
font-family: "JetBrains Mono", "SF Mono", "Monaco", "Consolas", monospace;
```

### Font Weights
- **Light**: 300 (for large display text)
- **Regular**: 400 (body text)
- **Medium**: 500 (emphasized text)
- **Semibold**: 600 (headings)
- **Bold**: 700 (strong emphasis)
